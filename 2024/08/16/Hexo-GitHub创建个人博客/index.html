Hexo+GitHub创建个人博客

目的

创建一个个人博客，记录自己的学习历程和一些个人信息。选择使用Hexo+GitHub创建个人博客

步骤

一、准备环境

1、node环境

首先，安装 nodejs， 因为Hexo是基于 Node.js 驱动的一款博客框架。
⭐node.js官网

2、git环境

然后，安装git, 一个分布式版本控制系统，用于项目的版本控制管理，作者是 Linux 之父。
⭐git官网
安装完成后通过cmd命令查看两者版本


以上情况即安装成功!

二、注册GitHub账号并创建仓库

1、注册

⭐GitHub官网

2、创建仓库


注意选择public
注意仓库的取名格式：用户名.github.io(这将是以后的访问域名)

到这里代表我们Github账号以及仓库都已经创建完毕 可以把下面这段仓库的地址复制下来留着后面配置时会用到（推荐使用ssh，https容易后期出现bug）

三、本地配置Hexo

1、新建一个文件夹

2、打开git命令窗口

3、安装Hexo

4、初始化Hexo

如果bash: hexo: command not found，则在前面加npx

(初始化成功后blog目录(文件夹)会多出许多文件)

输入npx hexo s启动本地服务器，用于浏览主题

四、设置SSH

1、生成SSHkey

首先在git下
输入命令：cd ~/.ssh进入.ssh文件
输入命令：ssh-keygen -t rsa -C ‘注册时的邮箱地址’

接着去查看C盘目录(此文件夹所在C:\用户\用户名目录下)，然后在记事本中打开id_rsa.pub文件并复制生成的key

2、将key配置到GitHub中

在GitHub主页的头像下找到seething

侧边栏选择SSH，然后新建一个key

输入信息点击增加

验证:

输入命令（git）：ssh -T git@github.com 并且输入yes之后，行末尾会显示你的用户名（绑定成功邮箱会收到邮件提醒）

接着在本地绑定与Github的用户名和邮箱（git）
输入命令：git config –global user.name “注册时用户名”
输入命令：git config –global user.email “注册时邮箱”

五、上传博客

1、打开blog文件夹中的_config.yml

2、将2-1中的仓库地址复制，然后将文章末尾改为

3、安装上传工具

4、预览

5、生成文件

6、部署到GitHub

之后每次上传博客都重复5、6两个步骤

部署之后可发现GitHub中多了文件，此时浏览器输入：用户名.github.io(Github仓库名)就可以访问博客啦（刚刚部署后需要一定时间加载，出现未更新等待一段时间即可）
