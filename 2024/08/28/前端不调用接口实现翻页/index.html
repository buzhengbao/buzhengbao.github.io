前端不调用接口实现翻页

问题

后端一次性给定全部数据，前端实现分页(vue3+elementplus)，翻页不调接口，描述实现逻辑。

实现方法

在 Vue 3 和 Element Plus 中实现前端分页逻辑，不需要每次翻页时调用接口，可以按照以下步骤进行：

接收数据: 从后端获取全部数据，并将其保存在 Vue 组件的 data 或 ref 中。

定义分页参数: 在组件中定义分页参数，例如当前页码 (currentPage)、每页显示的条数 (pageSize) 和总页数 (totalPages)。

计算当前页数据: 使用分页参数计算出当前页的数据。可以使用 slice 方法从原始数据中提取当前页的数据。

展示数据: 将计算出的当前页数据绑定到模板中，并展示。

分页控件: 使用 Element Plus 的分页组件 () 控制页码和每页条数，并响应页码变化。
